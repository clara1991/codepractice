专栏简介

喜欢读诗词，除了体会诗词本身的美感之外，诗词背后的故事更是我感兴趣的所在。本专栏通过探究和讲述诗词及其相关人物和历史的掌故，从而更好地理解诗词之美。

唐宋诗词，千古传诵。

字里行间，能赋深情。

人面桃花，十年一梦。

诗间偶拾，词林撷英。
